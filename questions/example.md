# 例題
## googleでクレスコを検索して結果画面を確認するテスト
  1. ブラウザを起動、Google検索サイトにアクセス
  2. 検索ボックスに「クレスコ」の文字を入力する
  3. Google検索ボタンを押下する
  4. ブラウザを閉じる
